Feature Flagging Demo

If you check the javascript of this page, you will see that this code has feature flags. When enabled or disabled, different features will become enabled or disabled.

  1. Choose a number from 1 to 9999.
  2. Pad it to 4 digits with leading zeros if needed.
  3. Arrange the digits in descending order to form the largest number.
  4. Arrange the digits in ascending order to form the smallest number.
  5. Subtract the smaller number from the larger number.
  6. Repeat the process with the result.

If all digits are the same, the result becomes 0000. Otherwise, repeating the steps will eventually reach 6174.
