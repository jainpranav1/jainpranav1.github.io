DNS Diagram

I have been studying DNS recently. I created a diagram using Mermaid below illustrating how DNS works. Check out the source code! 

            flowchart LR
                client[Client]
                client --www\.pranav.com IP?--> resolver[DNS Resolver]
                root[Root NS - 2.3.4.5]
                resolver --.com NS IP?--> root
                root --3.4.5.6--> resolver
                com[.com NS - 3.4.5.6]
                resolver --pranav.com NS IP?--> com
                com --4.5.6.7--> resolver
                pranav_com[pranav.com NS - 4.5.6.7 - Route 53]
                resolver --www\.pranav.com IP?--> pranav_com
                pranav_com --1.2.3.4--> resolver
                resolver --1.2.3.4--> client
                server[Web Server - 1.2.3.4]
                client --GET--> server
                server --HTML--> client
