Cloud Defense

In this tower defense game, you play as Rain, a new cloud architect for the startup Stratosphere Technologies. Your task is to develop and maintain the cloud infrastructure for the company's new QR code generator Zap. You must stay on your toes to keep up with varying traffic patterns without going bankrupt.

This game serves as a fun introduction cloud concepts such as:

  • scale in/out vs up/down
  • different traffic patterns
  • different types of cloud compute (ex. VM, function)
  • other cloud services for handling traffic (ex. queue, load balancer)

This game is partly inspired by the fun tower defense game Server Survival. Please go give the game a try!
